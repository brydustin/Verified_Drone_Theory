## Soundness warning on the UPhi Tier 2 sorry

Follow-up investigation found that the remaining `sorry`
`Phi_par_parallel_slot_F_eta_identification` is not merely an unfinished
technical proof.  As currently stated, it appears to ask Isabelle to prove the
wrong scalar identity for the definitions in this repo.

The repo uses the phase convention
`phase c x j = cis (-(c \<bullet> x$j))`, so in c-adapted coordinates
`z_j = exp(-i*kappa*u_j) = cos(kappa*u_j) - i*sin(kappa*u_j)`.  Therefore the
triple contribution to `b_1 = Im M_1` is `-u_j * sin(kappa*u_j)`.

On the gauge slice `b = 0`, the moment formula is

```text
Phi_1 = g_1(a^2 + b^2) + 2g(b_1 a - a_1 b)
      = a * (g_1*a + 2g*b_1).
```

Thus differentiating the natural `E_1 = g_1*a + 2g*b_1` in a parallel
`u_j` slot gives a scalar involving

```text
-g_1*kappa*sin(kappa*u_j)
-2g*(sin(kappa*u_j) + kappa*u_j*cos(kappa*u_j)).
```

This does not match the current `F_eta` target

```text
cos(kappa*u) - kappa*(u - eta)*sin(kappa*u),
```

whose `cos - kappa*u*sin` part is the derivative of `u*cos(kappa*u)`, i.e.
the real-part moment `a_1`, not the imaginary-part moment `b_1`.

Conclusion: do not spend reintegration effort trying to prove
`Phi_par_parallel_slot_F_eta_identification` as written.  Either the paper-side
u-slice scalar in this branch needs a convention/sign/index correction, or the
Isabelle Tier 2 theorem should be restated with the actual scalar function
generated by `Phi_par` under the repo's moment definitions.  The existing
`uphi_reduce_pointwise` theorem is only conditionally useful if that upstream
identity is replaced by a sound statement.
